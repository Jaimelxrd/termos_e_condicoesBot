Política de Privacidade — Bot de Anotações Pessoal

Última atualização: julho de 2026

Este aplicativo é um bot de uso estritamente pessoal e privado, desenvolvido e utilizado exclusivamente pelo próprio desenvolvedor para gerenciamento de anotações via WhatsApp (criação, listagem e exclusão de notas). Não é um serviço público e não está disponível para uso por terceiros.

Dados coletados

O aplicativo processa apenas as mensagens de texto enviadas pelo próprio desenvolvedor ao número de WhatsApp vinculado ao bot. Essas mensagens são usadas exclusivamente para interpretar comandos (ex: criar, listar ou excluir anotações) e são armazenadas em um banco de dados privado, hospedado pelo próprio desenvolvedor.

Uso dos dados

Os dados coletados são usados apenas para a funcionalidade do bot descrita acima. Nenhuma informação é analisada, vendida ou utilizada para fins publicitários.

Compartilhamento de dados

Nenhum dado é compartilhado com terceiros. O aplicativo se comunica apenas com a API oficial do WhatsApp (Meta) para envio e recebimento de mensagens.

Retenção e exclusão

Os dados permanecem armazenados enquanto o serviço estiver em uso. O desenvolvedor pode excluir qualquer anotação a qualquer momento através dos próprios comandos do bot.
